# SQL-Server-Power-BI-Dashboard-Project
Relational database built in SQL Server connected to Power BI dashboard using Excel datase


 📌 Project Overview
This project involves building a relational database in SQL Server 
using real-world sales data and visualizing it through an 
interactive Power BI dashboard.

 🎯 Objective
- Import and structure raw Excel data into SQL Server
- Create a relational database with multiple tables
- Connect SQL Server to Power BI
- Transform data using Power Query Editor
- Build an interactive dashboard for business insights

 🛠️ Tools & Technologies Used
| Tool | Purpose |
|------|---------|
| Microsoft Excel | Raw data source |
| SQL Server (SSMS) | Database creation and management |
| Power BI Desktop | Data visualization and dashboard |
| Power Query Editor | Data transformation and cleaning |

 🗄️ Database Tables
The database **"SQL Project"** contains the following tables:
- **Budgets2017** – Budget data for year 2017
- **Customers** – Customer details and information
- **Product** – Product catalog and details
- **Regions** – Regional/geographical data
- **SalesOrder** – Sales transactions and order data

 🔗 Project Workflow
1. Raw data prepared in Excel (`SQL_Dataset.xlsx`)
2. Data imported into SQL Server using SSMS
3. Tables structured and organized in the database
4. SQL Server connected to Power BI Desktop
5. Data cleaned and transformed in Power Query Editor
6. Relationships established in Power BI Model View
7. Interactive dashboard built with charts and KPIs

 📊 Dashboard Features
- Sales performance overview
- Region-wise analysis
- Product-wise breakdown
- Customer insights
- Budget vs Actual comparison
